About

This interactive map shows various entity types operating with AI in the UK. These have been classified by type of entity and, if they are a company, by sector. Entities are represented on the map by coloured dots. Dots of multiple colours indicate entities that fall into multiple categories.

How to use this map

Toggle entity types to display using the menus on the right.

Click on the dots to see the entity name and click on the name to open their website in a new tab (cmd / ctrl-click to stay on the map tab).

To zoom use the +/- buttons, double click, or shift drag to select an area.
